Car Management CLI (Java)

A Java console application that interacts with a Car Management REST API. This CLI allows users to manage cars, track fuel, and view fuel statistics without any external dependencies.

Features

Add new cars with brand, model, and year.

View a car by its ID.

View all cars.

Update car information.

Delete a car.

Add fuel for a car (liters, odometer reading, price).

View fuel statistics (total fuel, total cost, average consumption).

Fully interactive CLI menu.
